# AI Meeting Task Manager

AI-powered meeting notes → task extraction and management dashboard.

## Features

* Extract tasks from meeting notes
* Task management dashboard
* Assign tasks
* Track status

## Tech Stack

Frontend: React + Vite
Backend: Flask + SQLite

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m flask --app app run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
